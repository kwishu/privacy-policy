# プライバシーポリシー

Spotline（以下、「本アプリ」といいます）は、ユーザーの個人情報の保護を重要視し、以下の方針に基づき個人情報を取り扱います。

## 1. 収集する情報

本アプリでは、以下の情報を収集する場合があります。

### (1) アカウント情報
- ユーザー名
- メールアドレス
- プロフィール画像
- Apple Sign In を利用した場合の識別子（apple_subject）

### (2) 投稿データ
- ユーザーが投稿した写真（サーバー保存前にEXIF情報を自動削除。詳細は「写真のメタデータについて」を参照）
- 投稿内容（テキスト、タグ、公開設定）
- 訪問日時

### (3) 位置情報
- 投稿の位置情報（緯度・経度、都道府県・市区町村・国）
- ※投稿時のみ取得し、バックグラウンドでの追跡は行いません。
- ※設定により位置情報をぼかして表示する機能（FUZZY モード）があります。

### (4) 認証・デバイス情報
- APNs デバイストークン（デバイスを識別し、プッシュ通知の送信に使用）
- プッシュ通知設定（いいね・フォロー・コメント等の通知のオン/オフ）
- 最終ログイン日時・ログイン履歴

### (5) 課金・サブスクリプション情報
- Apple StoreKit 2 のトランザクションID（originalTransactionId）
- プレミアムプランの有効期限

### (6) ユーザーアクション情報
- フォロー・ブロック関係
- 通報・報告情報（何を・どのような理由で報告したか）

### (7) 技術情報（不正利用防止のため）
- IPアドレス・デバイスID（IDFV）（レート制限処理のためにリクエスト時に参照しますが、データベースへの保存は行いません）
- サーバーインフラが自動収集するアクセスログ（Google Cloud Run が管理）

## 2. 写真のメタデータについて

投稿写真には、撮影時のGPS座標・日時・撮影機種などのEXIF情報が含まれる場合があります。本アプリでは以下の処理を行います。

- **GPS座標・撮影日時の活用**: 写真のEXIF情報からGPS座標と撮影日時を抽出し、スポットの位置情報・訪問日時として保存します（ユーザーが手動で変更することも可能です）。
- **EXIFの削除**: 画像ファイル自体は、サーバーへのアップロード前にすべてのEXIF情報を除去して保存します。つまり、保存された画像ファイルからは撮影機種や正確なGPS座標などは読み取れません。

## 3. 情報の利用目的

収集した情報は、以下の目的で利用します。

- 本アプリのサービス提供および地図表示機能の提供
- ユーザー間のフォロー・コミュニティ機能の提供
- プッシュ通知の送信
- 不正アクセス、スパム行為、多重登録などの防止
- サービスの改善、不具合の調査、および品質向上
- プレミアムプランの管理・検証

## 4. 広告について

本アプリは Google AdMob を使用した報酬型広告を配信しています。広告を視聴することで、投稿制限の一時解除などの特典を受けられる場合があります。広告の表示・視聴に関する非個人情報（広告インタラクションなど）が AdMob の仕組みにより収集される場合があります。これらの情報は個人を特定するものではなく、クロスアプリトラッキングは行いません。詳細は [Google のプライバシーポリシー](https://policies.google.com/privacy) をご参照ください。

## 5. 運営によるデータの確認

本アプリでは、以下の場合において運営者がユーザーデータを確認することがあります。

- 利用規約違反の調査
- 不正利用やスパムの確認
- サービス運営上必要と判断される場合

これらの確認は、サービスの健全な運営を目的として必要な範囲でのみ行います。

## 6. データの管理

本アプリは、ユーザーの個人情報を適切に管理し、不正アクセス、紛失、改ざん、漏えいなどの防止に努めます。通信はすべてSSL/TLSにより暗号化されます。

## 7. 第三者への提供

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者へ提供することはありません。

- 法令に基づく場合
- 本人の同意がある場合
- サービスの提供に必要な範囲で、以下の外部サービスを利用する場合

| サービス名 | 用途 | 所在国 |
|---|---|---|
| Cloudflare R2 | 画像ストレージ | 米国 |
| Neon (PostgreSQL) | データベース | 米国 |
| Google Cloud Run | サーバー基盤 | 米国 |
| Google AdMob | 広告配信 | 米国 |
| Apple APNs | プッシュ通知 | 米国 |
| Resend | パスワードリセットメール送信 | 米国 |

上記サービスはいずれも十分な個人情報保護水準を有しており、適切な契約上の保護措置のもとでデータを提供します（個人情報保護法第24条に基づく対応）。

## 8. データの削除（退会）

ユーザーは、アプリ内の「アカウント削除」機能を利用して、いつでもアカウントを削除できます。
削除を実行した場合、以下のデータが即座に物理削除され、復旧することはできません。

- アカウント情報（ユーザー名・メールアドレス・プロフィール画像など）
- 投稿データ（スポット・おでかけ・画像ファイル）
- フォロー・ブロック・いいね・ブックマーク情報
- ログイン履歴・通知履歴

なお、不正利用防止のためにサーバーインフラ（Google Cloud Run）が自動収集するログについては、Google のデータ保持ポリシーに従い管理されます。

## 9. プライバシーポリシーの変更

本ポリシーは、必要に応じて変更される場合があります。
変更後のプライバシーポリシーは、本ページにて公開された時点から効力を生じるものとします。
